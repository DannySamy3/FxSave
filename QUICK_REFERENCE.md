## Manual Step-by-Step

### 1. Install Dependencies
```bash
npm install
cd python_model && pip install -r requirements.txt && cd ..
```

### 2. Train Model (First Time)
```bash
cd python_model && python train.py && cd ..
```

### 3. Generate Prediction
```bash
cd python_model && python predict.py && cd ..
```

### 4. Run App
```bash
npm run dev
```

### 5. View Dashboard
Open: **http://localhost:3000**

## Performance Tips

1. **Run predictions at market open** (9:30 AM ET)
2. **Use cron/Task Scheduler** to automate daily updates
3. **Monitor accuracy** - track correct vs incorrect signals
4. **Combine signals** - don't trade prediction alone
5. **Risk management** - always use stop losses
6. **Confidence filter** - only trade signals > 55%

## Common Customizations

### Change Prediction Update Frequency
Edit `pages/index.js`:
```javascript
// Change from 5 minutes (300000ms) to your preference
const interval = setInterval(fetchPrediction, 5 * 60 * 1000);
```

### Change Model Training Parameters
Edit `python_model/train.py`:
```python
model = XGBClassifier(
    n_estimators=100,      # Change number of trees
    max_depth=6,           # Change tree depth
    learning_rate=0.1,     # Change learning rate
    # ... other params
)
```

### Change Historical Data Range
Edit `python_model/train.py`:
```python
gold_data = download_gold_data(start_date='2018-01-01')  # Change date
```
